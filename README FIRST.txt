Aperture 2.0.0 RC2
====================

Start with QUICKSTART_WINDOWS.md or run "Install Aperture.cmd" from an extracted folder.

This is a release candidate for final validation. Back up important Aperture Libraries before testing. The application installation and the Aperture Library are separate; repair, upgrade, and uninstall must preserve the Library.

See RELEASE_NOTES.md for scope, known limitations, and deferred Release 3 work.
