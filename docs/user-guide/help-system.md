# Aperture Help System

Aperture includes an integrated, offline documentation browser. Choose **Help → User Guide…** to browse user, operations, project, and developer documentation without opening separate windows.

## Organized documentation browser

The contents pane groups documents by purpose:

- Getting Started
- Using Aperture
- Operations
- Reference
- Project
- Developer Documentation

Architecture, database, coding, plugin, charter, handover, and build documentation remain available because Aperture encourages informed extensions and specialised branches. They are grouped separately so ordinary users can focus on task-oriented guidance.

## Direct Help menu items

The Help menu keeps the most useful actions directly accessible:

- User Guide
- Vision
- What's New
- Troubleshooting
- Keyboard Shortcuts
- Help for Current Screen
- Check for Updates

**Keyboard Shortcuts** remains a direct menu item and is also available with `Ctrl+/`. Press `F1` for context-sensitive help.

## About Aperture

**About Aperture** remains a separate top-level menu item. Its current overview is the default page. Additional browsable pages provide Donate, licences and attribution, acknowledgements, and system information.

## Search and offline operation

The Help browser searches bundled Markdown content locally. No network connection is required.
