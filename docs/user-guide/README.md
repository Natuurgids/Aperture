# Aperture User Guide

## Library and Viewer

The Library workspace provides paged browsing and asynchronous thumbnails. Viewer supports fit, zoom, pan, metadata context, and navigation in the current result order. Originals are treated as immutable.

## Import and organization

Imports are planned before execution, resumable, and isolated per file. Exact duplicates are detected using content identity. Use collections and saved searches to organize photographs without moving originals.

## Observations and taxonomy

Confirmed observations remain distinct from AI suggestions. Observation History, species dashboards, timelines, evidence photographs, life lists, and statistics are derived from accepted records. Taxonomy packages retain their NatureAI_Next format.

## AI review

Aperture runs supported inference locally. Review screens expose model and score provenance, regional and seasonal context where available, and accept/reject/defer controls. Ecological context is explanatory and does not modify the underlying visual score.

## Activity and Settings

Long-running work appears in Activity Center and can be paused, cancelled, retried, or resumed where supported. Settings contains AI Resources, Regional Knowledge, Health Check, Branding & Project, and diagnostic information.

## Offline maps

See [Managing Offline Maps](offline-maps.md) for regional downloads, storage planning, `.apkg` bundle import, updates, and removal.
