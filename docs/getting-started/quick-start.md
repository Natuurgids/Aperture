# Aperture Quick Start

1. Start Aperture and create or open a library.
2. Import a small test folder before a large archive. Choose managed, referenced, or hybrid storage deliberately.
3. Browse the Library grid and open a photograph in Viewer.
4. Add ratings, tags, notes, location, or taxonomy through the Inspector.
5. Configure local AI resources under Settings before generating suggestions.
6. Review AI results explicitly; suggestions never replace confirmed metadata automatically.
7. Create a backup before large imports, taxonomy activation, or resource upgrades.

Aperture works offline during ordinary use. Network access is limited to explicit software, taxonomy, model, or regional-resource acquisition workflows.

## Install by double-clicking

Extract the full release and double-click `Install Aperture.cmd`. You do not need to type a PowerShell command for a standard FullAI installation.
